## 1. 페이지 컨셉

### 브랜드 무드

메모의 경험, 사람이 행하는 물리적인 필기의 감각을 조금 투과하여 브랜드 무드를 연출하고자 했다.  
포스트잇, 종이 질감, 인덱스 스티커 등 실생활에서 쓰는 메모/노트 요소에서 레퍼런스를 가져왔다.  
컨셉이 직접적으로 드러나기보다, 경험 안에 은은하게 녹아드는 방향을 목표로 한다.

### 페이지 코어

> "더 많은, 다양한 이야기들을 우린 이 공간에서 말하고 있으니 — 네가 스스로 찾으러 오라."

사용자가 취향에 맞는, 혹은 새로운 아티클을 스스로 발견하고 온보딩할 수 있도록 유도하는 것이 목표다.

### 디자인 원칙

- 의도적인 레벨링 없이 동등한 전개로 배치한다. (카드 크기 위계 없음)
- 정보를 먼저 던지지 않는다. 사용자가 행동했을 때 드러나게 한다.
- 마치 서점 진열대처럼 — 이미지와 제목만 보이고, 더 알고 싶을 때 상세가 나온다.

## 2. 레이아웃 구조

```
Sense Makers list
  ├── Header          (Fixed — 스크롤 시 고정)
  ├── Index           (Fixed — 우측 고정, 스크롤 시 고정)
  └── Page-Body       (스크롤 영역)
        ├── Page-Intro
        └── ContentList
```

- Header와 Index는 스크롤과 무관하게 화면에 고정된다.
- Page-Body만 세로 스크롤된다.

## 3. 콘텐츠 카드 (ContentList-Card)

### 기본 상태

- 이미지 + 제목 + 날짜만 노출
- 설명, 태그, 읽는 시간 등 부가 정보는 숨김 처리

### 호버 상태

- 이미지 위로 딤(어두운 오버레이) 적용
- 오버레이 위로 콘텐츠 등장

**호버 시 노출 정보**

| 항목 | 내용 |
|------|------|
| Tag | 카테고리 태그 |
| ReadTime | 읽는 시간 (예: 6분 읽기) |
| Description | 아티클 소개 3줄 (책 뒷면 형식, 별도 작성) |

### 인터랙션 명세 (피그마 미구현 — 개발 단 처리)

- 딤 : 이미지 위에 그라데이션 오버레이. 하단이 진하고 상단으로 갈수록 투명해지는 형태.
  - 예: `background: linear-gradient(to top, rgba(0,0,0,0.6) 0%, rgba(0,0,0,0) 100%)`
- 텍스트 등장 : 아래에서 위로 슥 올라오는 방식
  - `transform: translateY(12px) → translateY(0)`
  - `opacity: 0 → 1`
  - `transition: 200ms ease`
- 클릭 시 : 바로 아티클 페이지로 진입 (중간 단계 없음)

### 카드 구조 레이어명

```
ContentList-Card
  ├── Card-Image
  │     └── Card-Overlay
  │           └── Overlay-Content
  │                 ├── Overlay-Meta
  │                 │     ├── Meta-Tag
  │                 │     ├── Meta-ReadTime
  │                 │     └── Meta-Divider
  │                 └── Overlay-Description
  ├── Card-Info
  │     ├── Card-Title
  │     └── Card-Date
```

## 4. 카드 배열 (피그마 미구현 — 개발 단 처리)

- 2열 고정 그리드
- 카드 너비는 브라우저 너비에 따라 반응형으로 늘고 줄어든다.

```css
display: grid;
grid-template-columns: repeat(2, 1fr);
gap: [피그마 gap 값 참조];
```

- 카드 가로:세로 비율은 황금비 기반 — `1 : 1.618`
  - 예: `aspect-ratio: 1 / 1.618`
- 피그마 내에서는 대표 사이즈(1440px) 기준으로만 표현되어 있음.

## 5. 인덱스 탭 (Index-Tab)

### 역할

카테고리 필터. 누르면 해당 카테고리의 아티클만 노출된다.  
인덱스 스티커 형태로 페이지 우측에 고정 배치. 책 옆면의 인덱스 탭에서 착안했다.

### 카테고리

카테고리 목록은 현재 임시 상태이며 추후 확정 예정.  
색상은 브랜드 서브 컬러들로 구성하여 카테고리의 다양함을 표현할 예정.

### 상태별 표현

| 상태 | 표현 |
|------|------|
| 기본 | 옅은 색상 |
| 호버 | 기본보다 살짝 진해짐 |
| 활성화 | 카테고리 고유 색상으로 채움 |

- 텍스트 레이블은 항상 노출
- 탭 너비는 텍스트 길이에 따라 자연스럽게 달라짐 (의도된 편차)
- **다중 활성화 가능** — 여러 카테고리 동시 선택 지원

## 6. 미구현 항목 요약

| 항목 | 사유 | 처리 방법 |
|------|------|-----------|
